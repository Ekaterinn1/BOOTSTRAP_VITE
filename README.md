# Selle paketti juhis

## Alustades tööd
Igakordselt: alustades tööd puhtas
arvutis (seal ju teie proekti pole)
esimese asjana kloonite selle projekti.
`git clone repository`

## Peale kloonimist
`npm install`

## Käivitamine

`npm start`

## Juhul, kui projekt on juba arvutis olemas

`git pull`

git add .
